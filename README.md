# long-dev-search-suggestion

## Prerequisites

- python 3.10 + (Based on Owner's Environment)

## Get Started

```bash
$ pip install -r requirements.txt
```

```bash
$ flask db init
$ flask db migrate
$ flask db upgrade
$ python load_data.py
```
